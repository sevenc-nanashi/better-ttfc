# T-Two-F-C

東映特撮ファンクラブのPC版サイトをより便利にするためのユーザースクリプト。

> [!WARNING]
> このユーザースクリプトは非公式です。また、これによって発生した問題については一切の責任を負いません。利用は自己責任でお願いします。

## 機能一覧

- 作品・視聴ページ：
  - 古い作品（ブレイドなど）のタイトルに当時の番組表の表記（データソース：しょぼいカレンダー）を追加
  - ブラウザフルスクリーンモードを追加
  - 動画プレイヤーにキーボードショートカットを追加
    - スペースキー：再生/一時停止
    - 左右矢印キー：10秒巻き戻し/早送り
    - 上下矢印キー：音量アップ/ダウン
    - Mキー：ミュートの切り替え
    - Tキー：ブラウザフルスクリーンモードの切り替え
    - Fキー：フルスクリーンモードの切り替え

## インストール

1. [Tampermonkey](https://www.tampermonkey.net/)を入れます。
2. [ここ](https://raw.githubusercontent.com/sevenc-nanashi/t-two-f-c/built/index.user.js) をクリックして、Tampermonkeyのダッシュボードに追加します。

## 開発

依存関係をインストールするには、以下のコマンドを実行してください：

```bash
bun install
```

開発サーバーを起動するには、以下のコマンドを実行してください：

```bash
bun run dev
```
